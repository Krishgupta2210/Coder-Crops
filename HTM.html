Iot Based plant monitoring system

this device is very beneficial for plants and trees. As we go on vacation or go for work, we dont have time to take care of our plants. So It basically use the STM 32 board. There is a soil sensor in it which tells us about the soil moisture, there is a pir sensor which can detect any animal activity on plant, it has a dht11 sensor which can tell us about the temperature suitable for our plant. there is esp module in it which make it suitable for IOT this module make this product connected to an app called BLYNK
